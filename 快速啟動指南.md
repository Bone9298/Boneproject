# 🚀 快速啟動指南

## 📌 只要記住一個檔案！

**雙擊執行**：`🚀 啟動主選單.bat`

這是唯一需要的啟動器！所有功能都在裡面。

---

## 🗂️ 選單說明

### 【主要功能】

#### 1️⃣ 啟動本地系統 ⭐ **推薦日常使用**
- 使用本地 MariaDB + Python
- 速度快，功能完整
- 適合個人日常管理商品

**何時使用**：
- ✅ 日常管理商品
- ✅ 測試新功能
- ✅ 本地開發

---

#### 2️⃣ 啟動 Docker 系統
- 容器化環境
- 隔離、乾淨
- 準備部署前測試

**何時使用**：
- ✅ 準備部署到伺服器前測試
- ✅ 想要乾淨的獨立環境
- ⚠️ 注意：圖片載入較慢（44MB資料）

---

#### 3️⃣ Ngrok 分享設置
- 讓朋友透過網路訪問您的系統
- 提供公開網址

**包含功能**：
1. 安裝 Ngrok（首次使用）
2. 測試分享功能
3. 啟動分享
4. 配置 Chrome 擴充功能
5. 恢復本地配置

**何時使用**：
- ✅ 想讓朋友幫忙管理商品
- ✅ 臨時分享給同事使用
- ⚠️ 需保持電腦開機

---

### 【工具選單】

#### 4️⃣ 資料管理工具
- 匯入資料到 Docker
- 匯入資料庫（本地）
- 停止 Docker 服務

#### 5️⃣ 系統配置工具
- 配置擴充功能
- 恢復本地設定

#### 6️⃣ 查看使用說明
- 列出所有可用文檔

---

## 💡 常見使用情境

### 情境 1：日常使用
```
1. 雙擊「🚀 啟動主選單.bat」
2. 選擇「1」(啟動本地系統)
3. 開始管理商品
```

### 情境 2：分享給朋友
```
1. 雙擊「🚀 啟動主選單.bat」
2. 選擇「1」(啟動本地系統) - 先啟動服務
3. 選擇「3」(Ngrok 分享設置)
4. 選擇「3」(啟動 Ngrok 分享)
5. 複製網址給朋友
```

### 情境 3：準備部署伺服器
```
1. 雙擊「🚀 啟動主選單.bat」
2. 選擇「2」(啟動 Docker 系統)
3. 測試所有功能
4. 查看「DEPLOYMENT.md」部署指南
```
