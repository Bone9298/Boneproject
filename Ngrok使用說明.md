# 使用 Ngrok 分享系統給朋友

## 🌐 什麼是 Ngrok？

Ngrok 可以讓您的本地應用透過網路公開訪問，朋友可以通過網址使用您的系統。

## 🚀 使用步驟

### 1. 安裝 Ngrok

如果還沒安裝：
1. 訪問 https://ngrok.com/download
2. 下載 Windows 版本
3. 解壓縮到任意資料夾
4. 將 ngrok.exe 所在資料夾加入系統 PATH

**快速測試是否安裝**：
```bash
ngrok version
```

### 2. 啟動本地系統

**雙擊執行**：`啟動系統.bat`

確認這兩個服務正在運行：
- ✅ 前端：http://localhost:8000
- ✅ API：http://localhost:5000

### 3. 啟動 Ngrok 分享

**雙擊執行**：`啟動Ngrok分享.bat`

會開啟兩個 ngrok 視窗。

### 4. 獲取分享網址

在 **8000 端口的 Ngrok 視窗**中，找到類似這樣的內容：

```
Forwarding  https://abc123.ngrok-free.app -> http://localhost:8000
```

**複製** `https://abc123.ngrok-free.app` 這個網址。

### 5. 分享給朋友

把網址傳給朋友，他們就可以訪問了！

例如：
```
嘿！用這個網址可以管理商品：
https://abc123.ngrok-free.app
```

## 📝 重要說明

### ✅ 優點
- 朋友可以直接使用，不需要安裝任何東西
- 即時同步，您和朋友看到的都是同樣的資料
- 完全免費（免費版有限制）

### ⚠️ 注意事項

1. **保持電腦開啟**
   - 您的電腦必須保持開機且運行系統
   - 關機後朋友就無法訪問

2. **網址會改變**
   - 免費版每次重啟 ngrok，網址會改變
   - 需要重新告訴朋友新網址

3. **安全性**
   - 任何有網址的人都可以訪問
   - 只分享給信任的朋友
   - 不要公開到網路上

4. **速度限制**
   - 免費版有流量和連線數限制
   - 如果很多人同時使用可能會慢

5. **Ngrok 警告頁面**
   - 朋友首次訪問會看到 ngrok 警告頁面
   - 點擊 "Visit Site" 即可繼續

## 🛠️ 常見問題

### Q: 朋友說無法連接？
A: 檢查：
1. 您的系統是否還在運行？
2. Ngrok 視窗是否還開著？
3. 給朋友的是前端網址（8000端口）？

### Q: 如何停止分享？
A: 關閉 Ngrok 視窗即可，網址立即失效。

### Q: 可以固定網址嗎？
A: 免費版不行，付費版可以。

### Q: API 的網址要給朋友嗎？
A: **不用**！前端會自動處理，只給前端網址即可。

### Q: 多人同時使用會衝突嗎？
A: 不會，大家操作的都是同一個資料庫，會即時同步。

## 📱 使用 Chrome 擴充功能

如果朋友也想用 Chrome 擴充功能：

1. 修改 `extension/popup.html` 中的 API 網址：
   ```javascript
   const API_BASE_URL = 'https://your-api-url.ngrok-free.app/api';
   ```
   
2. 將整個 `extension` 資料夾分享給朋友

3. 朋友在 Chrome 中載入擴充功能（開發者模式）

## 🔒 進階：付費版功能

如需長期使用，可考慮 Ngrok 付費版：
- 固定的網址（不會改變）
- 移除警告頁面
- 更高的流量限制
- 自訂域名
